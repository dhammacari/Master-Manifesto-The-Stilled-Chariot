# Master-Manifesto-The-Stilled-Chariot
# The Stilled Chariot (ราชรถที่หยุดนิ่ง)

**"A Digital Manifesto for Global Peace based on Buddhist Ethics"**

โปรเจกต์เชิงพุทธศิลป์ดิจิทัล (Digital Dhamma Art) ที่นำคติธรรมจาก **"เทวสุรสงคราม"** มานำเสนอผ่านรูปแบบ Interactive Storytelling เพื่อสื่อสารถึงความตระหนักรู้ต่อผู้บริสุทธิ์ในวิกฤตสงครามโลกปัจจุบัน

### Core Message
- **Aims:** เพื่อย้ำเตือนว่าความเมตตาไม่ใช่ความอ่อนแอ แต่คือ "ความกล้าหาญ" ที่จะหยุดวงจรแห่งเวร
- **Philosophy:** นำเสนอหลัก "ขันติ" และ "กรุณา" ในรูปแบบ Minimalist

### Tech Stack
- HTML5 / CSS3 (Dark Mode)
- Vanilla JavaScript (Scroll-triggered animations)
